Welcome,

this is a quick introduction to help you get started using this API. On the left sidebar you see all available documentation resources. To access protected parts of the API you can read the authorization guide. On the support page you can see all available support options if you get stuck.

The API reference lists all available endpoints. It is the main entry point to explore the capabilities. All supported request methods are shown in a tab navigation. There you can see the request and response format. The format is a html representation of the JSON format. Each property refers to a property in the JSON object.
